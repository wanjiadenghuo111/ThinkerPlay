# Playground Taxonomy

The playground is organized by the kind of thinking each experiment supports.

## Areas

| Area | Focus | Example ideas |
| --- | --- | --- |
| `algorithms/` | Logic, strategy, and complexity | Sorting, pathfinding, recursion |
| `ai/` | Learning, inference, and representation | Perceptrons, CNNs, neural networks |
| `math/` | Abstract structures made visible | Fractals, geometry, probability |
| `physics/` | Natural rules and simulation | Particles, springs, waves |
| `systems/` | Complexity, hardware, and emergence | Storage, cellular automata, boids |

## Naming

Use readable, URL-friendly slugs:

```text
playground/ai/perceptron/
playground/algorithms/sorting/
playground/systems/cellular-automata/
```

Every experiment directory should contain an `index.html` and a `README.md`.
