📖 HDD 的 I/O 过程

当你要读取一个扇区（由 CHS 或 LBA 指定）时，硬盘控制器会按以下步骤操作：

步骤 1️⃣：磁头寻道（Seek）

• 磁头臂移动，将读写磁头定位到目标柱面（即目标磁道所在的半径位置）。

• 这个过程需要 几毫秒（典型值：2–10 ms），取决于起始位置和目标距离。

• 此时盘片仍在高速旋转，但磁头不在正确磁道上，无法读取目标数据。

步骤 2️⃣：盘片旋转（Rotational ）

• 磁头已停在目标磁道上方，但目标扇区可能刚转过去。

• 必须等待盘片继续旋转，直到目标扇区转到磁头正下方。

• 平均等待时间为 半圈时间，例如：

- 7200 RPM → 转一圈 = 8.33 ms → 平均旋转延迟 ≈ 4.17 ms

- 15000 RPM → 平均 ≈ 2 ms

步骤 3️⃣：数据传输（Transfer）

• 扇区经过磁头时，数据被读出（通常只需零点几毫秒）。

• 每个盘面对应一个磁头，所有的磁头都是连在同一个磁臂上的，因此所有磁头只能“共进退”，外内圈循环移动。

• 所有盘面中相同的磁道组成柱面，按当前柱面各磁头依次点亮预读取一批扇区进行缓存，再筛选目标扇区返回给文件系统。
